Быстрая сортировка, сортировка Хоара (англ. quicksort), часто называемая qsort (по имени в стандартной библиотеке языка Си) — алгоритм сортировки, разработанный английским информатиком Тони Хоаром во время его работы в МГУ в 1960 году. Один из самых быстрых известных универсальных алгоритмов сортировки массивов: в среднем O(n\log n) обменов при упорядочении n элементов; из-за наличия ряда недостатков на практике обычно используется с некоторыми доработками. 

			    
                  algorithm quicksort(A, lo, hi) is
                    if lo < hi then
                      p:= partition(A, lo, hi)
                      quicksort(A, lo, p)
                      quicksort(A, p + 1, hi)
					  
                  algorithm partition(A, low, high) is
                    pivot:= A[(low + high) / 2]
                    i:= low
                    j:= high
                    loop forever
                      while A[i] < pivot 
                        i:= i + 1
                      while A[j] > pivot
                        j:= j - 1
                      if i >= j then
                        return j
                      swap A[i++] with A[j--]
