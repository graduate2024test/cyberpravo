Описание: Программы на языке Assembler из примеров EX_1, EX_2, EX_3 выводят на экран фамилию, имя и номер группы и завершаются.

Пример EX_1

			  
				stak segment stack 'stack'      
				db 256 dup (?)                  
				stak ends                       
				data segment 'data'             
				Hello db 'Yakunin Vladimir 351 CSIT!$' 
				data ends                       
				code segment 'code'             
				assume CS:code,DS:data,SS:stak  
													
				start:                          
				mov AX,data                     
				mov DS,AX                       
				mov AH,09h                      
				mov DX,offset Hello             
				int 21h                         
				mov AX,4C00h                    
				int 21h                         
				code ends                       
				end start

				mov AX,4C00h

				end start
